# 在庫管理システム（Excel VBA × MySQL）

## 📦 概要
ExcelからMySQLに接続し、商品マスタの情報を操作できる業務用ツールです。

## 🔧 使用技術
- Excel VBA（ADODB接続）
- MySQL（ローカル環境）
- ODBCドライバ（MySQL ODBC 9.3）

## 🔁 主な機能
- 商品一覧の取得（SELECT）
- 単価の更新（UPDATE）
- データベースとの接続・切断
